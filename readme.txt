== Description ==
Woo Special Offers plugin allows adding other products as offers.


== Installation ==

1. Upload `woo-special-offers` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Enable/Disable the offers option from "Special Offers" tab in "WooCommerce" settings submenu

4. In product edit page, add the offer details in "Special Offer" tab in Product Data Section.
